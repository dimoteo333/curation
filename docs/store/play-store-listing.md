# Play Store Listing

## 앱 이름

큐레이터

## 짧은 설명

개인 기록을 기기 안에서 분석해 나만의 통찰을 전하는 프라이빗 AI 큐레이션

## 전체 설명

큐레이터는 흩어진 개인 기록을 한곳에 모아, 지금의 나에게 필요한 맥락과 통찰을 한국어로 정리해 주는 개인 큐레이션 서비스입니다.

일정, 메모에서 내보낸 텍스트, 일기, 회고, 업무 기록처럼 나를 설명하는 자료는 많지만, 시간이 지나면 다시 꺼내 보기 어렵습니다. 큐레이터는 이런 기록을 기기 안에서 읽고 연결해, 질문에 맞는 맥락을 찾아 주고 스스로를 돌아보는 데 도움이 되는 답변을 제공합니다.

가장 중요한 원칙은 프라이버시입니다. 큐레이터는 개인 기록이 기본적으로 기기 안에 머물도록 설계되었습니다. 텍스트 데이터는 로컬 저장소에 보관되며, 민감한 개인 텍스트 필드는 앱 계층에서 보호됩니다. 사용자는 설정 화면에서 데이터 원본 상태를 확인하고, 필요하면 전체 데이터를 한 번에 삭제할 수 있습니다.

큐레이터의 핵심 경험

- 온디바이스 중심 처리: 가능한 한 기기 내부에서 검색과 큐레이션을 수행해 개인 데이터 노출을 줄입니다.
- 개인 기록 연결: 텍스트 파일과 최근 일정 같은 개인 맥락을 묶어, 단편적인 기록 대신 흐름을 보여 줍니다.
- 한국어 큐레이션 UX: 질문, 요약, 후속 제안까지 한국어 사용자 경험에 맞춰 설계했습니다.
- 개인 지식 회수: “요즘 왜 이렇게 지쳤지?”, “최근 내가 자주 놓친 패턴은 뭐지?” 같은 질문에 기록 기반의 실마리를 제공합니다.
- 명확한 상태 표시: 현재 기기에서 네이티브 AI 경로를 쓰는지, 폴백 경로를 쓰는지 앱 안에서 확인할 수 있습니다.

큐레이터는 단순한 메모 앱이 아닙니다. 쌓아 둔 기록을 다시 찾기 쉽게 만들고, 스스로를 이해하는 질문에 답하도록 돕는 개인용 큐레이션 레이어입니다. 업무 회고를 정리하는 직장인, 생활 패턴을 돌아보는 학생, 감정의 흐름을 기록하는 사용자 모두에게 적합합니다.

또한 큐레이터는 과장된 자동화보다 신뢰할 수 있는 개인 맥락 회수에 집중합니다. 답변은 기록에서 찾은 근거와 함께 구성되며, 사용자가 다음에 어떤 질문을 던지면 좋을지 후속 제안도 제공합니다. 즉흥적인 생성보다, 내 기록을 바탕으로 한 정리와 연결을 우선합니다.

이런 분께 추천합니다.

- 흩어진 메모와 일정, 텍스트 기록을 한데 모아 보고 싶은 분
- 개인 데이터가 외부 서버로 전송되는 것이 부담스러운 분
- 한국어 기반으로 나를 위한 회고와 큐레이션을 받고 싶은 분
- 파일 기반으로 차분히 자기 기록을 축적하고 다시 활용하고 싶은 분

큐레이터는 나를 더 많이 기록하게 만드는 앱이 아니라, 이미 기록한 나의 흔적을 더 잘 이해하게 만드는 앱입니다. 개인 정보 보호를 우선하면서도, 온디바이스 AI와 개인 검색을 통해 나만의 맥락을 다시 발견해 보세요.

## 피처 그래픽 설명

- 1024x500 기준
- 중앙 메시지: "내 기록에서 찾는 오늘의 통찰"
- 배경: 차분한 크림 톤과 종이 질감, 카드형 기록 조각, 일정 배지, 검색 하이라이트
- 강조 요소: 프라이버시, 온디바이스 AI, 한국어 개인 큐레이션

## 스크린샷 필요 화면

- 온보딩 화면
- 홈 화면 첫 진입
- 질문 입력 및 큐레이션 결과 카드
- 설정 화면의 런타임 상태 배지
- 파일 가져오기 화면 또는 파일 선택 후 상태
- 캘린더 동기화 상태 화면
- 데이터 삭제 확인 흐름
