# App Store Listing

## 앱 이름

큐레이터

## 부제

기기 안에서 완성하는 개인 큐레이션

## 설명

큐레이터는 개인 기록을 다시 읽고 연결해, 지금의 나에게 필요한 통찰을 한국어로 정리해 주는 온디바이스 중심 개인 큐레이션 앱입니다.

메모, 텍스트 기록, 일정 같은 자료는 쌓일수록 가치가 커지지만, 정작 필요할 때는 흐름을 놓치기 쉽습니다. 큐레이터는 사용자가 남긴 기록을 기기 안에서 정리하고 검색해, 질문에 맞는 맥락을 찾아 보여 줍니다. 단편적인 메모를 다시 찾는 수준을 넘어, 최근의 패턴과 감정, 반복되는 주제를 스스로 이해하도록 돕습니다.

큐레이터가 중요하게 보는 기준은 세 가지입니다.

첫째, 프라이버시.
개인 기록은 가능한 한 기기 내부에 머물러야 합니다. 큐레이터는 온디바이스 경로를 기본값으로 두고, 로컬 저장 데이터를 중심으로 동작합니다. 민감한 개인 텍스트는 보호된 저장 경로를 사용하며, 사용자는 설정에서 가져온 데이터 상태를 확인하고 언제든 삭제할 수 있습니다.

둘째, 개인 맥락.
큐레이터는 기록을 단순히 모아 두지 않습니다. 일정, 텍스트 파일, 메모에서 내보낸 기록처럼 서로 다른 출처의 정보를 개인 맥락으로 엮어 질문에 답합니다. 예를 들어 최근 피로감, 반복된 업무 패턴, 자주 언급한 감정 같은 흐름을 기록 기반으로 되짚을 수 있습니다.

셋째, 한국어 사용자 경험.
큐레이터의 질문 응답, 요약, 후속 제안은 한국어 사용 환경에 맞춰 설계되었습니다. 결과는 단순 검색 결과 목록이 아니라, 읽기 쉬운 요약과 다음 행동으로 이어질 수 있는 후속 질문 형태로 제공됩니다.

이 앱은 이런 분께 적합합니다.

- 개인 메모와 일정, 텍스트 기록을 한곳에서 다시 활용하고 싶은 분
- 외부 서버 업로드 없이 개인 데이터를 다루고 싶은 분
- 기록을 남기는 것에서 끝나지 않고, 스스로를 위한 해석과 연결이 필요한 분
- 한국어 기반의 차분한 회고 경험을 원하는 분

큐레이터의 핵심 기능

- 온디바이스 중심 개인 검색
- 개인 기록 기반 질문 응답
- 텍스트 파일 가져오기와 최근 일정 반영
- 현재 런타임 경로와 데이터 상태 확인
- 전체 데이터 삭제를 포함한 명확한 제어권 제공

큐레이터는 생산성을 과장하는 앱이 아니라, 이미 남겨 둔 나의 기록을 더 의미 있게 다시 읽도록 돕는 앱입니다. 개인적인 메모, 일정, 생각의 흔적을 기기 안에서 안전하게 연결하고, 나만의 맥락으로 정리해 보세요.

- 큐레이션 결과 상세
- 설정의 프라이버시 및 런타임 상태
- 파일 가져오기
- 캘린더 가져오기

## 앱 개인정보 처리 레이블

- 데이터 수집: 기본적으로 수집 없음
- 기기 내 처리 데이터:
  - 사용자 콘텐츠: 메모 텍스트, 가져온 텍스트 파일, 개인 기록
  - 일정 정보: 사용자가 명시적으로 허용한 최근 캘린더 이벤트
  - 진단 정보: 기본 수집 없음
- 추적:
  - 사용자 추적 안 함
- 데이터 연결:
  - 기본 배포 구성에서는 사용자 데이터를 제3자 데이터와 연결하지 않음
- 데이터 사용 목적:
  - 앱 기능 제공
  - 개인 기록 기반 큐레이션 생성
  - 기기 내 검색 및 회수
- 사용자 제어:
  - 설정에서 데이터 원본 상태 확인 가능
  - 전체 로컬 데이터 삭제 가능
