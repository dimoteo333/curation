# Changelog

## 0.1.0 - Sprint 1 to Sprint 10

- Sprint 1: 온디바이스 런타임 상태 UI, 개발자 패널, 네이티브 브릿지 가드레일을 추가했다.
- Sprint 2: 한국어 의미 임베딩 폴백과 로컬 검색 캐시를 도입해 검색 품질을 개선했다.
- Sprint 3: 온보딩, 설정 화면, 파일 import 파이프라인을 추가해 개인 기록 인제스트를 시작했다.
- Sprint 4: 기본 모바일 UX와 로컬 큐레이션 흐름을 정리해 개인 질문-응답 경험을 다듬었다.
- Sprint 5: 로컬 벡터 검색과 응답 생성 흐름을 안정화해 온디바이스 기본 경로를 강화했다.
- Sprint 6: Android 네이티브 브릿지와 release shrinker 구성을 정리하고 실제 빌드 검증을 추가했다.
- Sprint 7: 로컬 저장 데이터 암호화, secure storage 키 관리, delete-all 흐름으로 프라이버시 보호를 강화했다.
- Sprint 8: 문서, 계약, 검증 가드레일을 정리해 운영 전 점검 체계를 보강했다.
- Sprint 9: 최근 캘린더 import, import history, deduplication, 데이터 소스 상태 UI를 추가했다.
- Sprint 10: Android/iOS 릴리스 준비, 스토어 메타데이터, 버전 관리, release CI/CD 파이프라인을 추가했다.
